3) Table Tag(테이블태그)

table : 테이블 태그를 작성하기 위해서 가장 바깥 영역에 입력하는 태그

tr : table row 의 약자 - 테이블의 행을 정의할 때 사용하는 태그 

th : table header 의 약자 - 각 행 혹은 각 열의 대표성을 띄고있는 셀을 지칭 하고자 할 때

td : table data 의 약자 - 각 행 혹은 각 열의 일반 보통의 셀을 지칭할 때

*HTML5 에 적용된 문법 (시멘틱 태그 = semantic tag - 의미를 부여하는 태그)
=> 사람인, 잡코리아 취업 지원할때 ( HTML5 / 시멘틱태그 준수!! ) 이런게많음
thead / tbody / tfoot : 각 table 의 영역을 구분짓고 싶을때 사용하는 태그

*figure : figcaption을 사용하기 위해서 사용하는 tag
*caption : table tag를 사용할 때, 해당 테이블의 제목을 입력하고자 할 때 사용 - 수평정렬
*figcaption : table 태그를 사용할때, 테이블의 제목을 입력할 수 있음 - 좌측정렬

*colgroup : 열을 그룹핑 하고자 할때 사용하는 tag
( col : 실제 colgroup 안에 있는 각각의 열을 지칭하고자 할때)

*colspan : 열을 병합하고싶을때
*rowspan : 행을 병합하고싶을때

*row : 행 / column : 열

*Style 적용에 대한 우선순위
1. thead / tbody / tfoot
2. colgroup > col
