1. 컴퓨터에게 우측상단 토글버튼이 존재한다 라는것을 알려줘야함
- JS  => 변수(*바구니) 선언 => 토글 버튼을 담아준다

2. 컴퓨터에게 gnb 네비게이션이라는 존재역시 알려줘야함
- JS => 변수 선언 => gnb 네비게이션을 담아준다.

3. 우측상단 토글버튼을 클릭하지 않은 상태일 때는 gnb 네비게이션이 나타나면 안된다.
- CSS => 특정요소의 존재 자체를 없앨 수 있는 스타일 속성
(display = none)


4. 우측상단 토글버튼을 클릭했을 때 , gnb 네비게이션이 나타난다
- JS => 이벤트 (*User가 어떤 액션 행동을 취했을 때 ) => 행위를 만들어라 (이벤트 => 리액선을 만들어라!!!) addEventListener 라는 함수 (* 어떤 기능을 정의해놓는 역할)을 사용할 수 있음.
